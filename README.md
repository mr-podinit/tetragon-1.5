# Setup

## Configure k0s

Copy k0s/config.yaml in /etc/k0s, create it.

```bash
sudo mkdir -p /etc/k0s/
sudo cp config.yaml /etc/k0s/
sudo k0s install controller  --single -c /etc/k0s/config.yaml
k0s 
```
